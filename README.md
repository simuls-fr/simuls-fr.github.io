# Simulateur de Retraite

Ce dépôt contient une application React (Vite + Tailwind CSS) qui permet de simuler la retraite par capitalisation.

## Installation

```bash
npm install
```

## Lancement en développement

```bash
npm run dev
```

Ouvrez ensuite http://localhost:3000 dans votre navigateur.

## Construction

```bash
npm run build
```
